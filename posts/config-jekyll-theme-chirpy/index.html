使用主题

参考：jekyll-theme-chirpy/README.zh-CN.md

  1. 使用 starter template 来快速创建 Jekyll 站点
  2. 克隆仓库到本地
  3. 修改 _config.yml 配置文件并提交（提交后将触发 GitHub Actions workflow 在 gh-pages 分支生成站点）
  4. 设置 GitHub 仓库，将 gh-pages 设置为 GitHub Pages 的发布源
